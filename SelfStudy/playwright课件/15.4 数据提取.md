### 15.4.1 获取元素的文本
使用 `inner_text()` 方法可以获取指定元素的文本内容。例如，如果你想要获取页面中 ID 为 `box` 的元素的文本内容，可以这样做：

```python
text = page.locator('#box').inner_text()
print("元素的文本内容：", text)
```

这里，`page.locator('#box')` 用于定位 ID 为 `box` 的元素，然后通过调用 `.inner_text()` 方法来提取该元素内的纯文本内容。

### 15.4.2 获取所有匹配元素的文本内容
如果你需要获取多个匹配元素的文本内容，并希望将它们以列表形式返回，可以使用 `all_inner_texts()` 方法。比如，获取页面中所有链接（`<a>` 标签）的文本内容：

```python
texts = page.locator("a").all_inner_texts()
print("所有匹配元素的文本内容：", texts)
```

这行代码会找到页面上的每一个 `<a>` 标签，并收集它们各自的文本内容到一个列表中。

### 15.4.3 获取匹配元素的属性值
若要获取某个特定元素的一个属性值，比如图片的 `src` 属性，可以使用 `get_attribute()` 方法。以下是如何获取页面上第一个 `<img>` 元素的 `src` 值的例子：

```python
src = page.locator('img').get_attribute('src')
print("图片的src属性值：", src)
```

这段代码首先定位到页面中的第一个 `<img>` 标签，然后请求其 `src` 属性的值。

### 15.4.4 获取 `<input>` 元素的 `value` 属性值
对于 `<input>` 类型的表单控件，如果想直接读取其当前的值，可以使用 `input_value()` 方法。如下所示，获取 ID 为 `user` 的输入框的当前值：

```python
value = page.locator('#user').input_value()
print("input元素的value属性值：", value)
```

这特别适用于需要检查用户输入或预填充值的情况。

### 15.4.5 获取元素的位置
当你需要知道某个元素在网页上的确切位置时，可以利用 `bounding_box()` 方法。它返回一个字典，其中包含了元素相对于视口的坐标（`x`, `y`）以及它的宽度 (`width`) 和高度 (`height`)。下面是如何查询 ID 为 `box11` 的元素的位置信息：

```python
rect = page.locator('//*[@id="box11"]').bounding_box()
print("元素的位置：", rect)
```

这里，`rect` 变量将会存储一个字典，格式类似于 `{'x': 500, 'y': 200, 'width': 80, 'height': 60}`，表示元素左上角位于 (500, 200) 坐标处，宽80像素高60像素。

### 15.4.6 统计匹配到的元素个数
最后，如果你想计算页面上有多少个特定类型的元素，可以使用 `count()` 方法。例如，统计页面中有多少个 `<li>` 列表项：

```python
count = page.locator('li').count()
print("匹配到的元素个数：", count)
```

此方法返回一个整数，代表页面上与给定选择器相匹配的所有元素的数量。这对于验证页面结构或者进行自动化测试非常有用。
